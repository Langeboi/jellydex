Kommer Snart

Viser digitale filmutgivelser og neste episode for innhold som er forespurt i Seerr.

Server-til-server-adressen, for eksempel http://10.10.100.3:5050
Finnes i Seerr → Settings → General. Nøkkelen sendes aldri til nettleseren.
ISO-landkode brukt for digitale filmdatoer, for eksempel NO, DK, SE eller US.
Filmer bruker kun TMDB release type 4 («Digital») i valgt region. Kinodato brukes aldri. TMDB skiller ikke sikkert mellom abonnementstrømming og digital leie/kjøp; manglende dato vises derfor som ukjent.
